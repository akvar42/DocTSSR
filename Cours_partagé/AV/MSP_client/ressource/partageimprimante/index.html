    Installation et partage de l'imprimante HP LaserJet M9050 MFP

    1. Démarrage du gestionnaire d'ajout d'imprimante :
    2. On se rend dans Panneau de configuration > Matériel et audio > Périphériques et imprimantes.

    3. On clique sur Ajouter une imprimante.

    4. Choix du type d'imprimante :

    5. Lorsqu'on est interrogé sur la manière d'installer l'imprimante, on choisit Ajouter une imprimante en utilisant une adresse IP ou un nom d'hôte.

    6. On clique sur Suivant.

    7. Renseignement de l'adresse IP :

    8. Dans le champ Nom d'hôte ou adresse IP, on inscrit 172.16.63.251.
    9. On s'assure que l'option Interroger l'imprimante et sélectionner automatiquement le pilote à utiliser est activée.

    10. On clique sur Suivant.

    11. Sélection du pilote :

    12. Si Windows reconnaît l'imprimante, il peut proposer un pilote par défaut. Sinon, on sélectionne HP comme fabricant et trouve LaserJet M9050 MFP dans la liste des pilotes.
    13. Si le pilote n'est pas listé, il faudra peut-être le télécharger depuis le site officiel de HP et l'installer manuellement.

    14. On clique sur Suivant.

    15. Nom de l'imprimante :

    16. On donne un nom à l'imprimante, par exemple : "HP LaserJet M9050 MFP".

    17. On clique sur Suivant.

    18. Options de partage :

    19. Quand l'option de partage est présentée, on coche Partager cette imprimante.
    20. Dans le champ Nom de partage, on indique un nom qui sera utilisé sur le réseau pour accéder à cette imprimante. Par exemple, "HPM9050_Shared".

    21. On clique sur Suivant.

    22. Finalisation :

    23. Une page récapitulative est présentée. On vérifie toutes les informations.

    24. On clique sur Terminer.

    25. Test :

    26. Pour confirmer que tout est bien configuré, on imprime une page test.

    27. Accès depuis un autre poste :

    28. Depuis un autre ordinateur du réseau, on va dans Périphériques et imprimantes, puis on ajoute une imprimante.
    29. On recherche l'imprimante partagée par nom et l'ajoute à la liste des imprimantes de l'ordinateur.
